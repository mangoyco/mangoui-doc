# ImgPreview 图片预览 pass 0.0.13+

# 基础用法

渲染的效果和原生的 <img/> 是一样的，原生的自定义属性也可以直接使用

展开查看
<mg-img-preview src="/xxx/mango.png" title="点击预览" />
1

# 可操作的预览

ops 选项可以对预览的图片进行一些简易的操作如放大，缩小等

展开查看
<mg-img-preview :ops="true" src="/xxx/mango.png" title="点击预览" />
1

提示

可以使用 disabled 属性停用预览的功能

# ImgPreview Attributes

参数说明类型可选值默认值
src图片路径boolean--
ops是否展示默认的预览操作栏boolean-false
disabled是否禁用预览功能boolean--

# ImgPreview Event

事件说明回参
close结束预览时触发（点击遮罩层）-
closed结束预览时触发（动画结束）-
关于ImgPreview Event

目前的版本只开放了关闭时的事件，开启时的事件可以直接通过绑定click事件完成
暂时未支持开启动画结束时的钩子函数...
