    # Backtop Attributes

    参数 说明 类型 可选值 默认值
    right 距离右侧的距离 number - 30
    bottom 距离底部的距离 number - 30
    target 滚动目标的选择器 string - body
    appendBody 是否渲染在body标签 boolean - false
    visibleTop 滚动至该高度时显示 number - 0
