# 👀 介绍

Mango UI 是一个可以用于学习、搭建个人项目、博客的易用UI库。

基于Vue 3开发, 无其他外部依赖。

目前由于时间等原因，暂未开放代码，后续将开源于github。

考虑到部分同学（包括本人）上网成本问题，将同步于gitee 👀

如果在使用上出现问题或者有其他想法😜，欢迎与我联系 (opens new window)，十分荣幸。

也欢迎一起学习 😊

# 🔥 开发背景

  1. 个人开发与维护
  2. 坐标青岛，开发时间基本是“课余时间”，外加偶尔上线的时候闲下来会敲一敲 😜
  3. 开发目的主要是学习vue3

PS:

个人认为目前在UI库上，成熟与靠谱的选择还是蛮多的 （Element，AntDesign等的），当然如果有同学比较喜欢尝试，或者mangoui真的一不小心帮上了忙，个人表示非常荣幸，一定尽力为您解决问题 😊
